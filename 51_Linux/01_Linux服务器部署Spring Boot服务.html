0. 工具

使用 FinalShell 连接Linux服务器

1. 安装JDK

1.1 下载对应系统 JDK安装包

1.2 创建 JDK安装目录

1
cd /usr/
2
mkdir java
3
cd java

1.3 将 JDK安装包上传到服务器

1.4 解压 JDK安装包

xxxxxxxxxx
1
1
tar -zxvf jdk-8u281-linux-x64.tar.gz

1.5 配置环境变量

xxxxxxxxxx
1
1
vi /etc/profile

按 i 进入编辑模式，在文件最后加上

xxxxxxxxxx
4
1
JAVA_HOME=/usr/java/jdk1.8.0_281
2
PATH=$PATH:$HOME/bin:$JAVA_HOME/bin
3
export JAVA_HOME
4
export PATH

按ESC退出编辑模式

输入:wq保存并退出

1.6 使用source /etc/profile配置生效

xxxxxxxxxx
1
1
source /etc/profile

1.7 检查环境变量是否生效

xxxxxxxxxx
1
1
java -version

到此，JDK安装完成，环境变量已生效。

2. 安装MySQL

2.1 安装前，可以先检测系统是否已安装了 MySQL

xxxxxxxxxx
1
1
rpm -qa | grep mysql

2.2 如果系统已安装，可以进行卸载

xxxxxxxxxx
1
1
rpm -e --nodeps mysql

2.3 安装 MySQL

xxxxxxxxxx
1
1
wget http://repo.mysql.com/mysql57-community-release-el5-7.noarch.rpm

xxxxxxxxxx
1
1
rpm -ivh mysql57-community-release-el5-7.noarch.rpm

xxxxxxxxxx
1
1
yum update

xxxxxxxxxx
1
1
yum install mysql-server

2.4 设置权限

xxxxxxxxxx
1
1
chown mysql:mysql -R /var/lib/mysql

2.5 初始化 MySQL

xxxxxxxxxx
1
1
mysqld --initialize

2.6 设置权限

xxxxxxxxxx
1
1
chmod -R 777 /var/lib/mysql

2.7 启动 MySQL

xxxxxxxxxx
1
1
systemctl start mysqld

2.8 查看 MySQL 运行状态

xxxxxxxxxx
1
1
systemctl status mysqld

启动成功

2.9 查看版本信息

xxxxxxxxxx
1
1
mysqladmin --version

2.10 查看初始化密码

xxxxxxxxxx
1
1
cat /var/log/mysqld.log | grep password

2.11 连接 MySQL服务器

2.12 修改密码

xxxxxxxxxx
1
1
set password for root@localhost = password('root');

2.13 建库建表导入数据

我们将数据文件放置到/usr/mysql/目录下

xxxxxxxxxx
4
1
create database test;
2
use test;
3
set names utf8mb4;
4
source /usr/mysql/test.sql;

导入成功

3. 部署Spring Boot服务

3.1 在服务器上创建Spring Boot服务放置目录

xxxxxxxxxx
3
1
cd /usr/
2
mkdir test
3
cd test

3.2 上传Spring Boot服务到该目录

3.3 启动服务

xxxxxxxxxx
1
1
java -jar test-1.0.0.1.jar
